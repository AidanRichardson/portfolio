- **Version Control:** Git & GitHub Actions (CI/CD)

## Development
To run locally:

```bash
# Clone the repo
git clone https://github.com/AidanRichardson/portfolio.git

# Navigate into the directory
cd portfolio

# Install dependencies
npm install

# Run the development server
npm run dev
```
